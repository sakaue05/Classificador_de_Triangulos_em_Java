# Classificador de Triângulos em Java

Este projeto foi desenvolvido para praticar lógica de programação e fundamentos de Java.  
A aplicação recebe três valores, verifica se eles podem formar um triângulo e exibe o tipo correspondente.  
É um projeto simples, mas excelente para treinar validação, organização de código e conceitos básicos de POO.

---

## O que este projeto faz

- Valida se os três lados podem formar um triângulo
- Identifica se é o triângulo 3-4-5, um caso clássico usado em matemática e computação  
- Identifica automaticamente o tipo:
  - Equilátero  
  - Isósceles  
  - Escaleno  
  - Retângulo (verificação com Teorema de Pitágoras)
- Estrutura organizada em classes separadas  
- Código limpo e didático, fácil de entender

---

## O que eu pratiquei aqui

- Estruturas condicionais e validações  
- Criação de métodos e modularização  
- Organização de classes em um projeto Java  
- Pensamento lógico voltado à análise geométrica  
- Entrada, processamento e saída de dados no console  

Este é o tipo de exercício que ajuda a reforçar bases sólidas em programação, essenciais para evoluir para projetos maiores.
